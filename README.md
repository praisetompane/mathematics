# what is mathematics?

def: a formal language with precise and unambiguous definitions of its words(+,-,\*, π,e, etc.) and grammar.¹

**Disclaimer**: This is an ongoing and incomplete project to unpack these concepts.


# Langlans Program "continents" of mathematics²

* Number Theory
* Harmonic Theory
* Geometry

* Number Theory: Explains numbers
* Harmonic Theory: Explains Harmonics/Waves
* Geometry: Explains the intrinsic shapes of mathematical objects

# Mathematics Branches(Incomplete and Still Growing):³
* Pure Mathematics
    * Foundations
    * Algebra
    * Analysis
    * Geometry and Topology

    * Foundations: studies the formulation and analysis 
                    of the language, 
                    axioms 
                    logical methods
                on which all Mathematics is built/rests.
        * Logic and Model theory
        * Computability Theory and Recursion theory
        * Set Theory
        * Category Theory

    * Algebra: studies mathematical symbols/objects and the rules for manipulating them.
        * Group Theory
        * Ring Theory(includes Elemetary Algebra)
        * Field Theory
        * Module Theory
        * Galois Theory
        * Number Theory
        * Combinatronics
        * Algebraic Geometry
        
    * Analysis: studies with limits and related theories. Usually in the context of Real and Complex numbers and funtions.
        * Real Analysis and Measure Theory(Includes Elementray Calculus)
        * Complex Analysis
        * Tensor and Vector Analysis
        * Differential and Integral Equations
        * Numerical Analysis
        * Functional Analysis and Theory of Functions

    * Geometry and Topology
        Geometry: studies with concepts such as space, aread and volume
        Topology: studies the qualititative properties of space.

        * Euclidian Geometry
        * Non-euclidian Geometry
        * Absolute Geometry
        * Metric Geometry
        * Projective Geometry
        * Affine Geometry
        * Discrete Geometry and Graph Theory
        * Differential Geometry
        * General Topology
        * Algebraic Topology

* Applied Mathematics: studies the application of mathematical methods to various fields(e.g. Physics, Biology, Finance, etc}
        
    * Probability Theory
    * Statistics
    * Computer Science
    * Mathematical Physics
    * Game Theory
    * Systems and Control Theory
