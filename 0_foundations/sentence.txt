def sentence:   
    a statement with no free variables.
        in a given universe
            it MUST be True or False (Kunen, 2007:4).
                
    NB: The statement is True or False for the whole universe.

    Example:
        ∀x[x > 0 ⟹ ∃!y[y² = x ∧ y > 0]] (Kunen, 2007:4):
            Value       Universe
                True         ℕ
                False        ℝ

References:
    Kunen, K. 2007. The Foundations of Mathematics.
