def logical: using reason (Cambridge)

def deduction: the process of drawing conclusions from 
                    premises and syllogisms (Wolfram MathWorld).
    def premise(Q: axiom?): a statement that is assumed to be True (Wolfram MathWorld).
    def syllogism(rule of inference): a formal logical scheme
                                            used to draw a conclusion
                                                from a set of premises (Wolfram MathWorld).
