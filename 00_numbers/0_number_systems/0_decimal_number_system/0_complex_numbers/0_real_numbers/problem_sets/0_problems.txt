True or False?

  13. -4 <= -4:       True, Because they are on the same place on the number line

  14. -5 > -2:        False, Because -5 is further to the Left on the number line
                                     -2 is further to the Right on the number line

  15. -20 < -12:      True, Because -20 is further to the Left on the number line
                                     -12 is further to the Right on the number line

  16. 30.5 > 30.05:   True, Because 30.5 is further to the Right on the number line
                                    30.05 is further to the Left on the number line

  17. -4 < -4:        False, Because they are on the same place on the number line, they are Equal

  18. -71.24 > -71.2: False, Because -71.2 is further to the Right on the number line
