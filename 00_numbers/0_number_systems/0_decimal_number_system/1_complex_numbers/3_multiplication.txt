multiplication:
    plain english: multiply the two complex numbers as algebraic expressions (Censor, 2015).
    formal:
        (a+bi)(c+di)
            = ac + adi + bci + bdii
            = ac + adi + bci + bdi^2
            = ac + adi + bci + (bd * -1)
            = ac + adi + bci + (-bd)
            = ac + adi + bci - bd
            = ac - bd + adi + bci : grouped like terms
            = ac - bd + (ad+ bc)i: algebraic simplification

References:
    Censor, A. Algebra 1M - international. 05 - Complex numbers. 2015. Technion - International School Of Engineering.
