def cancellation with respect to addition and multiplication(Censor, 2015):
    - formal:
        if a + b = a + c, then b = c
            cancellation with respect to addition.
        if a × b = a × c, then b = c (a ≠ 0)
            cancellation with respect to multiplication.

References:
    - Censor, A. 2015. 04 - More properties of fields, Algebra 1M - international. Technion - International School Of Engineering.
